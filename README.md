# media-controller-rasp

## Project setup
```
npm install
```
---
## Production

Uses a pre-built version saved on `dist` directory

## Start app
```
npm run start
```
## Build app
```
npm run build
```
---
## Development usage

Requires 2 terminal sessions

### Real time compiles and hot-reloads for development
```
npm run serve
```

### Start app on development
```
npm run start-dev
```
